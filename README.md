# 🌱 AI Plant Disease Detection ChatBot

An AI-powered web application that detects plant diseases from leaf images using Deep Learning and provides disease diagnosis, treatment recommendations, and preventive measures through an interactive chatbot.

---

## 📖 Overview

Plant diseases can significantly reduce crop yield and quality. This project leverages Computer Vision and Deep Learning to identify plant diseases from leaf images and provide users with actionable insights.

The application integrates a **React.js frontend**, **FastAPI backend**, and a **PyTorch-based deep learning model** for accurate, real-time disease prediction.

---

## ✨ Features

- 🌿 Upload plant leaf images for disease detection
- 🤖 AI-powered disease classification
- 📊 Real-time prediction with confidence score
- 💊 Disease diagnosis and treatment recommendations
- 🛡️ Preventive measures for plant health
- 💬 Interactive chatbot interface
- 📱 Responsive and user-friendly web application

## 🧠 Model Information

- **Framework:** PyTorch
- **Model Format:** SafeTensors
- **Task:** Plant Disease Classification
- **Input:** Plant Leaf Image
- **Output:**
  - Predicted Disease
  - Confidence Score
  - Treatment Recommendations
  - Prevention Tips

---

## 📊 Dataset

The model was trained using the **PlantVillage Dataset** along with additional curated plant disease images to improve generalization and prediction accuracy.

---

## 🔄 Workflow

1. Upload a plant leaf image.
2. The image is processed by the FastAPI backend.
3. The deep learning model predicts the disease.
4. The chatbot displays:
   - Disease Name
   - Confidence Score
   - Disease Description
   - Treatment Recommendations
   - Preventive Measures
