🌿 About This Project

Plant Disease Detection System - An AI-powered solution for agricultural diagnostics

Features

  • 🖼️ Image-based disease detection using CNN
  • 📝 Text-based symptom analysis
  • 💊 Comprehensive treatment recommendations
  • 📊 Real-time statistics tracking
  • 💾 Offline data storage & export
  • 🌙 Dark mode support

Technology Stack

  • Backend: FastAPI, Python
  • ML Models: PyTorch, Transformers
  • Frontend: Vanilla JavaScript, HTML5, CSS3
  • Storage: Browser LocalStorage

Developer Information

Created by: Vanshika

Version: 2.0.0

Release Date: October 2025

License: All Rights Reserved

Model Information

  • Image Model: Custom CNN (38 disease classes)
  • Text Model: Transformer-based classifier
  • Dataset: New Plant Diseases Dataset (Augmented)
  • Accuracy: High confidence predictions
Note: This system is designed for educational and research purposes. For critical agricultural decisions, please consult with certified plant pathologists.
