1. Project Overview

The Plant Disease Detection System is an AI-powered web application that helps farmers and researchers identify plant diseases through two methods:

  • Image-Based Detection: Upload a photo of a plant leaf → CNN model identifies the disease
  • Text-Based Detection: Describe symptoms in natural language → Transformer model predicts the disease

For each detected disease, the system provides comprehensive treatment recommendations, prevention methods, and organic solutions.

Key Numbers: 38 disease classes | 14 plant species | 2 ML models | 100% local processing (no external APIs)

↓

↓
Click "Analyze Image" → POST /api/image-prediction
↓
Backend: PIL opens image → converts to RGB

4.3 Text Prediction Flow

User types symptom description
↓
Click "Analyze Symptoms" → POST /api/text-prediction
↓
Backend: HuggingFace pipeline("text-classification")
↓
DistilBERT tokenizer → model inference
↓
Output: LABEL_X (numeric ID) + confidence score
↓
LabelEncoder.inverse_transform → disease name
↓
Lookup recommendations (disease_recommendations.json)
↓
Return: { predicted_disease, confidence, recommendations }

5. ML Models — Architecture & Data Flow

5.1 Image Model — Custom CNN

PropertyDetail
Architecture3 Convolutional Blocks + Fully Connected Layers
Input224 × 224 RGB image (normalized)
Conv Block 1Conv2d(3→16, 3×3) → ReLU → MaxPool2d(2×2)
Conv Block 2Conv2d(16→32, 3×3) → ReLU → MaxPool2d(2×2)
Conv Block 3Conv2d(32→64, 3×3) → ReLU → MaxPool2d(2×2)
PoolingAdaptiveAvgPool2d(7×7)
FC LayersFlatten → Linear(3136→256) → ReLU → Dropout(0.5) → Linear(256→38)
Output38 class probabilities (via Softmax)
Normalizationmean=[0.4759, 0.5003, 0.4266], std=[0.2102, 0.1888, 0.2262]
Training DatasetNew Plant Diseases Dataset (Augmented)
Weight File Size3.3 MB

5.2 Text Model — DistilBERT

PropertyDetail
ArchitectureDistilBertForSequenceClassification
Base ModelDistilBERT (6 layers, 768 dim, 12 attention heads)
InputNatural language text (max 512 tokens)
TokenizerWordPiece (30,522 vocabulary)
Output15 class labels (LABEL_0 to LABEL_14)
Label Decodingscikit-learn LabelEncoder (.joblib file)
ActivationGELU
Dropout0.2 (classification), 0.1 (attention)
Model File Size268 MB (model.safetensors)
Note: The image model classifies into 38 classes (all plant+disease combinations). The text model classifies into 15 classes (disease types only). The LabelEncoder maps the text model's numeric output IDs back to human-readable disease names.

10. How to Run

Quick Start

Double-click start.bat — it auto-starts the server and opens the browser.

Manual Start

  1. Open Command Prompt or PowerShell
  2. cd "c:\Users\vansh\Downloads\plant diease project"
  3. .venv\Scripts\python.exe backend\main.py
  4. Open http://localhost:8000 in your browser
  5. Wait for loading screen to dismiss (models take ~30-60 seconds to load)
Requirements: Python 3.11 | All dependencies installed in .venv via pip install -r backend/requirements.txt
